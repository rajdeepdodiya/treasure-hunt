# treasure-hunt
Geochacing App developed for the Toronto Geocaching Club using React Native.
This helps users to create an account, view the available list of caches nearby,view caches within 10Km on map, view details of each cache and add new cache.

Users are able to create account, login to account and they can mark it as remember me if they prefer.
Users can view the map with nearby locations of caches, and if the map moves, it changes coordinates to find the nearby locations of caches with the new region.
Users can add cache which can be viewed by other users
